# **LIB-N-DRINK**

[About Project]  
LIB-N-DRINK는 학교 도서관 지정석 예약 시스템 및 음료 주문 시스템입니다. 

[기능]  
1. 회원가입 및 로그인을 통해 회원제 운영 가능
2. 도서관 지정석 자리 예약 및 취소
3. 음료 주문 및 결제 시스템
4. 관리자 모드를 통해 사용자 오류 및 실수를 제어합니다.
